## 目標詳述 (Goal Details)

### [G-001] 核心產品開發 (MVP Version)

#### 1. 核心願景
設計一個高品質的放置類網頁遊戲，讓玩家體驗從凡人到成仙的過程。

#### 2. 關鍵指標 (KPI)
- **通關時長：** 完全不手動干預下，達成結局需控制在 **120 分鐘 (±10分鐘)**。
- **多樣性：** 具備隨機性變量，確保每次遊玩過程不同。
- **結局數量：** 總共 **64 種不同結局**，基於屬性偏好、事件選擇與隨機數決定。

#### 3. 遊戲性要求
- **數值平衡：** 離線/在線產出平衡，數值曲線需經驗證避免崩壞。
- **境界系統：** 支援至少 5 個境界（練氣、筑基、金丹、元嬰、化神）。

#### 4. 技術架構
- **Frontend:** React + Tailwind CSS + Lucide Icons.
- **State:** React Hooks + LocalStorage.
